Есть TreeSet чисел, нужно отсортировать его в обратном порядке

Task 2
Есть TreeSet<Student>. Заполните его списком студентов. Если получили исключение, вспомните про интерфейс Comparable или Comparator

Task 3
Есть неупорядоченный склад с фруктами (допустим это List<Fruits>).
У фрукта есть название и кол-во в кг и тип (например, цитрусовые, косточковые, тропические)
Нужно произвести переучет:
- сгруппировать фрукты по типу
- посчитать весь каждого из фруктов
- отсортировать фрукты по названию

Task 4
Есть TreeSet имен, необходимо выбрать все от "H" до "W" (regex (регэксп))
