Task 1
Создайте класс Employee, Work. У Work есть атрибуты название и минимальный стаж.
У Employee есть имя, возраст и работа (work).
Запишите объект Employee в файл, затем восстановите его обратно в объект.

Task 2
Написать функцию, которая бы обходила весь каталог (включая вложенные папки и файлы) и печатала бы на экран структуру.
Это может быть папка с вашим проектом

Task 3
Написать программу, которая считает сколько раз в тексте MS Word документа встретится слово.
Документ и выбор слова на ваше усмотрение.
