Класс Carplay (находится рядом с файлом домашнего задания).

Создайте таблицу car_play и заполните ее данными

Задание:

***Task 2***
Создать класс, который будет перехватывать все исключения в проекте, выводить на консоль ошибку,
а пользователю возвращать сообщение "Sorry, something goes wrong, please try again later".
Затем запустите приложение и остановите контейнер с базой данных:

1. запустите приложение и остановите контейнер с базой данных:
- docker ps
- Копируете CONTAINER ID запущенного контейнера с БД
- docker stop [CONTAINER ID]

2. Выполните запрос, в котором участвует БД через Postman и убедитесь, что ваш перехватчик ошибок работает
3. Запустите остановленный контейнер с БД:
- docker start [CONTAINER ID]
4. Убедитесь, что приложение снова работает без ошибок.
